<h1 align="center">Create Web3 App 🚀🚀🚀</h1>

## 介绍

- 可快速搭建一个规范化的 web3 前端项目模板
  1. 包含基本 web3 方法，如钱包连接、文本签名、712 签名、网络切换等
  2. 使用 Eslint + Prettier 代码规范
  3. 使用 husky 管理代码提交
- 后端、合约开箱即可联调 approve 功能

## 快速开始

1. 安装依赖

```bash
npm install
# or
yarn
```

2. 启动本地开发服务

```bash
npm run dev
# or
yarn dev
```

## 注意事项

项目缺少 env 配置文件，如需联调 proxy approve 业务，请联系开发者！！！【仅搭建前端项目可忽略】
